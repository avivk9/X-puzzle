# X-puzzle

A solver for NxN tile puzzle in AI course, implemented using different algoritems

### EXAMPLES FOR MOVES:

    R - RIGHT:
    1 2 3       1 2 3
    4 0 5   =>  0 4 5
    6 7 8       6 7 8
    
    L - LEFT:
    1 2 3       1 2 3
    4 0 5   =>  4 5 0
    6 7 8       6 7 8

    U - UP:
    1 2 3       1 2 3
    4 0 5   =>  4 7 5
    6 7 8       6 0 8
    
    D - DOWN:
    1 2 3       1 0 3
    4 0 5   =>  4 2 5
    6 7 8       6 7 8
